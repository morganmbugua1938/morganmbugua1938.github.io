3. YouTube OAuth Token (only setup needed)

  1. Go to developers.google.com/oauthplayground
  2. In left panel, find "YouTube Data API v3" and select youtube.force-ssl
  3. Click "Authorize APIs" → sign in with your Google/YouTube account
  4. Click "Exchange authorization code for tokens"
  5. Copy the Access Token (starts with ya29...) → paste in Pipeline tab
  6. ⚠️ Token expires every hour — repeat when needed
